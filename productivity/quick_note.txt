REM Quick Note Creator
REM Creates a timestamped note file on desktop

DELAY 1000
GUI r
DELAY 500
STRING notepad
DELAY 500
ENTER
DELAY 1500
STRING Note - 
DELAY 200
STRING %date% %time%
DELAY 500
ENTER
ENTER
STRING 
DELAY 500
CTRL s
DELAY 1000
STRING %USERPROFILE%\Desktop\Note_
DELAY 200
STRING %date:~-4,4%%date:~-10,2%%date:~-7,2%_%time:~0,2%%time:~3,2%%time:~6,2%
DELAY 500
ENTER
DELAY 500
